# 09 - DevOps and Observability

Master the practices and tools that enable continuous delivery, deployment, and monitoring of modern software systems.

## What You'll Learn

- CI/CD pipelines
- Containerization (Docker, Compose)
- Kubernetes basics
- Monitoring and observability tools

## Weekly Breakdown

**Week 1: CI/CD Pipelines**
- GitHub Actions, GitLab CI, Jenkins

**Week 2: Containerization**
- Docker, compose files, multi-stage builds

**Week 3: Kubernetes Basics**
- Pods, services, deployments, Helm

**Week 4: Monitoring**
- Prometheus, Grafana, ELK, Loki, Sentry

---

By the end, you'll be able to build, deploy, and monitor robust systems.
