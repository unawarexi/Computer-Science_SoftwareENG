# 06 - Networking and Security

This module covers the essentials of networking and security for software architects. You'll learn how to design secure, reliable systems and APIs.

## What You'll Learn

- Protocols and web communication
- API design (REST, GraphQL, gRPC)
- Security essentials (XSS, CSRF, SQLi, JWT, OAuth2, TLS)
- Secure architecture design

## Weekly Breakdown

**Week 1: Protocols & Web Communication**
- HTTP/HTTPS, TCP/IP, WebSockets

**Week 2: API Design**
- REST, GraphQL, gRPC
- Building secure APIs

**Week 3: Security Essentials**
- XSS, CSRF, SQLi, JWT, OAuth2, TLS

**Week 4: Secure Architecture Design**
- Threat modeling
- Principle of least privilege, RBAC, secure storage

---

By the end, you'll be able to design and secure modern software systems.
