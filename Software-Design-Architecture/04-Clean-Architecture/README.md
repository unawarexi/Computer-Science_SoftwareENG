# 04 - Clean Architecture

This module introduces you to the principles and practices of Clean Architecture. You'll learn how to structure systems for maintainability, testability, and scalability.

## What You'll Learn

- Clean Architecture concepts
- Layered architectures: Hexagonal, Onion, Ports & Adapters
- Dependency inversion in multi-layer apps
- Applying SOLID at the architecture level

## Weekly Breakdown

**Week 1: Introduction to Clean Architecture**
- Key concepts and terminology
- Why Clean Architecture matters

**Week 2–3: Layered Architectures**
- Hexagonal, Onion, Ports & Adapters
- Building modular projects

**Week 4: Dependency Rule**
- Understanding and applying the dependency rule
- Using SOLID in architecture

---

By the end, you'll be able to design systems that stand the test of time.
