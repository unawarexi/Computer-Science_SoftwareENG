# 12 - Soft Skills for Architects

Technical skills are only half the battle. This module focuses on the communication, leadership, and documentation skills every architect needs.

## What You'll Learn

- Writing architecture decision records (ADRs)
- Effective communication and mentorship
- Leadership and stakeholder management

## Weekly Breakdown

**Week 1: Documentation**
- Writing ADRs, design docs, decision logs

**Week 2: Communication**
- Communicating tradeoffs, constraints, and risk

**Week 3: Mentorship**
- Leading code reviews, onboarding, pair programming

**Week 4: Leadership & Influence**
- Managing stakeholders, negotiating scope, vision alignment

---

By the end, you'll be ready to lead teams and projects as an architect.
