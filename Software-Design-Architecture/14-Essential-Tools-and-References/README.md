# 14 - Essential Tools and References

Familiarize yourself with the tools and references every architect should know. This module is your go-to resource for diagrams, documentation, and modeling.

## What You'll Learn

- Diagramming and modeling tools (Draw.io, Lucidchart, PlantUML, Mermaid, etc.)
- Documentation tools (Notion, Confluence, Markdown, etc.)
- API design and database modeling tools
- How to choose the right tool for the job

## Topics Covered

- Diagramming: Draw.io, Lucidchart, Visio, Archimate, PlantUML, Mermaid, C4-Model Tools, Structurizr, yEd, OmniGraffle, Graphviz
- Modeling: Archi, Modelio, Enterprise Architect
- Documentation: Notion, Confluence, Markdown, Google Docs, Docusaurus
- Decision Records: ADR Tools
- Database Design: dbdiagram.io
- Whiteboarding: Miro, Whimsical
- Wireframing: Figma, Balsamiq, Pencil Project
- API Design: Swagger/OpenAPI, Stoplight
- BPMN Modeling: Camunda Modeler
- Visualization: Piktochart

---

By the end, you'll know which tools to use for every architecture task.
