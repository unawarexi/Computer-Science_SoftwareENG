# Inter-Process Communication (IPC)

IPC is a mechanism that allows processes to communicate and synchronize their actions. It is essential for resource sharing, data exchange, and event notification in multitasking systems.

## IPC Methods:
- **Pipes:** Unidirectional/bidirectional communication channels.
- **Message Queues:** Processes send and receive messages via queues.
- **Shared Memory:** Multiple processes access a common memory region.
- **Semaphores:** Used for signaling and synchronization.
- **Sockets:** Communication over a network or between unrelated processes.

IPC must ensure data consistency, avoid race conditions, and provide security and access control.
