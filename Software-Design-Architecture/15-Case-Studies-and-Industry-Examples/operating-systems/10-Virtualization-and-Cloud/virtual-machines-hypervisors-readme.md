# Virtual Machines and Hypervisors

- **Virtual Machine (VM):** Emulates a physical computer, running its own OS and applications.
- **Hypervisor:** Software that creates and manages VMs. Types:
  - Type 1 (bare-metal): Runs directly on hardware (e.g., VMware ESXi, Xen).
  - Type 2 (hosted): Runs on a host OS (e.g., VirtualBox, VMware Workstation).

Hypervisors provide isolation, resource management, and flexibility.
