# Network Stack in OS

The network stack is a layered architecture that manages network communication. Common layers:

- **Application Layer:** User-facing protocols (HTTP, FTP).
- **Transport Layer:** Reliable/unreliable data transfer (TCP, UDP).
- **Network Layer:** Routing and addressing (IP).
- **Data Link Layer:** Physical addressing, error detection (Ethernet, Wi-Fi).
- **Physical Layer:** Transmission of raw bits over hardware.

The OS implements and manages these layers for networked applications.
