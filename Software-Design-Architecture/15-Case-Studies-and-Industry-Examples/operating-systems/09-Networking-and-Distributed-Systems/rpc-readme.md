# Remote Procedure Call (RPC)

RPC is a protocol that allows a program to execute procedures on a remote system as if they were local calls.

- **Stubs:** Client and server use stubs to marshal/unmarshal data.
- **Transparency:** Hides network communication details from the programmer.
- **Use Cases:** Distributed systems, microservices, client-server applications.

RPC simplifies distributed application development.
