# Distributed OS Concepts

A distributed operating system manages a collection of independent computers and makes them appear as a single coherent system.

- **Transparency:** Hides distribution from users (location, migration, replication, concurrency).
- **Resource Sharing:** Shares hardware and software resources.
- **Fault Tolerance:** Continues operation despite failures.
- **Scalability:** Supports growth in users and resources.

Examples: Amoeba, Plan 9, Google File System.
