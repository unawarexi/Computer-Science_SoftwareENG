# Buffering, Caching, and Spooling

- **Buffering:** Temporary storage of data during transfer between devices or processes. Smooths speed differences.
- **Caching:** Storing frequently accessed data in fast memory for quick retrieval (e.g., disk cache).
- **Spooling:** Staging data for devices that cannot accept interleaved data streams (e.g., print spooling).

These techniques improve I/O efficiency and system performance.
