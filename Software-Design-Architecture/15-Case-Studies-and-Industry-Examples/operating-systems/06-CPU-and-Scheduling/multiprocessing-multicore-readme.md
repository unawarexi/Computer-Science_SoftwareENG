# Multiprocessing and Multicore

## Multiprocessing
- Use of two or more CPUs within a single computer system.
- Increases throughput and reliability.
- Can be symmetric (SMP) or asymmetric (AMP).

## Multicore
- A single CPU chip contains multiple processing cores.
- Each core can execute instructions independently.
- Improves parallelism and energy efficiency.

Modern OSes are designed to exploit both multiprocessing and multicore architectures for better performance.
