# Real-Time Scheduling

Real-time scheduling is used in systems where tasks must be completed within strict timing constraints (e.g., embedded, industrial, medical systems).

- **Hard Real-Time:** Missing a deadline is catastrophic.
- **Soft Real-Time:** Occasional deadline misses are tolerable.

Common algorithms: Rate Monotonic Scheduling (RMS), Earliest Deadline First (EDF).

Real-time OSes guarantee predictable response times.
