# RAID and Redundancy

RAID (Redundant Array of Independent Disks) is a technology for combining multiple disks to improve performance and/or reliability.

- **RAID 0:** Striping, no redundancy, improved speed.
- **RAID 1:** Mirroring, redundancy, improved reliability.
- **RAID 5/6:** Striping with parity, balances speed and redundancy.
- **RAID 10:** Combines mirroring and striping.

Redundancy ensures data availability in case of hardware failure.
