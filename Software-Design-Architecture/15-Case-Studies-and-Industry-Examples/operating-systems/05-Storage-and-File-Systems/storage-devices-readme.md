# Storage Devices

Storage devices are hardware components used to store data persistently. Common types include:

- **Hard Disk Drives (HDD):** Magnetic storage, large capacity, slower access.
- **Solid State Drives (SSD):** Flash memory, faster access, more expensive per GB.
- **Optical Drives:** Use lasers to read/write data (CD, DVD, Blu-ray).
- **Flash Drives:** Portable, use NAND flash memory (USB sticks, SD cards).

Key concepts: access time, transfer rate, reliability, and durability.
