# Virtual Memory

Virtual memory is a technique that allows execution of processes that may not be completely in physical memory. It provides the illusion of a large, contiguous memory space by using disk storage as an extension of RAM.

## Key Concepts:
- **Demand Paging:** Pages are loaded into memory only when needed.
- **Page Replacement:** When memory is full, the OS swaps out pages using algorithms like FIFO, LRU, or Optimal.
- **Thrashing:** Excessive paging activity that degrades performance.

Virtual memory enables multitasking, process isolation, and efficient memory utilization.
