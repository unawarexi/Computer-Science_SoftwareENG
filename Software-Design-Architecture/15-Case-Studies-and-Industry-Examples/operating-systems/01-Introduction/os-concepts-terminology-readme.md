# Key OS Concepts and Terminology

- **Process:** Program in execution.
- **Thread:** Smallest unit of CPU execution.
- **Interrupt:** Signal to the CPU to stop current activity and execute a handler.
- **System Call:** Interface for user programs to request OS services.
- **Context Switch:** Saving and restoring process state during multitasking.
- **Deadlock:** Situation where processes wait indefinitely for resources.
- **Virtual Memory:** Abstraction of large memory using disk storage.
