# Types of Operating Systems

- **Batch OS:** Executes batches of jobs with minimal user interaction.
- **Time-Sharing OS:** Allows multiple users to interact with the system simultaneously.
- **Real-Time OS:** Guarantees response within strict time constraints (used in embedded, industrial systems).
- **Embedded OS:** Designed for specialized devices (IoT, appliances).
- **Mobile OS:** Optimized for smartphones and tablets (Android, iOS).
- **Desktop OS:** General-purpose, user-friendly (Windows, macOS, Linux).
- **Server OS:** Manages network resources and services (Windows Server, Linux Server).
