# Boot Process

The boot process initializes hardware and loads the operating system kernel into memory. Steps include:

1. **Power On:** CPU starts executing firmware (BIOS/UEFI).
2. **POST:** Power-On Self-Test checks hardware.
3. **Bootloader:** Loads OS kernel from disk.
4. **Kernel Initialization:** Sets up memory, devices, and system processes.
5. **User Space Launch:** Starts user-level services and login prompt.
