# 08 - Microservices and Monoliths

Explore the pros and cons of monolithic and microservices architectures. Learn how to design, split, and orchestrate services for real-world applications.

## What You'll Learn

- Monolith vs microservices
- Designing microservices (bounded contexts, DDD)
- Service orchestration (API gateway, service mesh)
- Hands-on project: splitting a monolith

## Weekly Breakdown

**Week 1: Monolith vs Microservices**
- Pros, cons, and case studies (Uber, Amazon, Netflix)

**Week 2: Designing Microservices**
- Bounded contexts, domain-driven design, inter-service communication

**Week 3: Service Orchestration**
- API gateway, service mesh, Consul, Istio

**Week 4: Hands-on Project**
- Split a monolith into services using Docker Compose

---

By the end, you'll know when and how to use microservices or monoliths.
