# 01 - Foundations of Architecture Thinking

Welcome to the Foundations of Architecture Thinking! This module sets the stage for your journey to becoming a software architect. Here, you'll develop the mindset, principles, and foundational knowledge that underpin all great architecture work.

## What You'll Learn

- The difference between coding and architecting
- The architect's mindset and responsibilities
- Core design principles (SOLID)
- Object-Oriented vs Functional paradigms
- Developer craftsmanship (KISS, DRY, YAGNI, Law of Demeter)

## Weekly Breakdown

**Week 1: Introduction & Mindset**
- What is software architecture?
- The architect's role in a team
- Key readings and video resources

**Week 2: Basic Design Principles**
- SOLID principles (Single Responsibility, Open/Closed, Liskov, Interface Segregation, Dependency Inversion)
- Practice exercises

**Week 3: OOP & Functional Paradigms**
- Compare OOP and FP
- Clean Code basics
- Common design patterns

**Week 4: Developer Craftsmanship**
- KISS, DRY, YAGNI, Law of Demeter
- Refactoring and code quality

---

By the end of this module, you'll have a strong foundation for all advanced topics ahead!
