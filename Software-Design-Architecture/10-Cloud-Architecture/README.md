# 10 - Cloud Architecture

Learn how to design, deploy, and manage systems in the cloud. This module covers the essentials of cloud platforms, serverless, and high-availability design.

## What You'll Learn

- AWS/GCP basics
- Serverless vs serverful architectures
- High availability and cost-effective design

## Weekly Breakdown

**Week 1: Intro to AWS/GCP**
- IAM, EC2, S3, Lambda, RDS, CloudFront

**Week 2: Serverless vs Serverful**
- Build and compare both deployment types

**Week 3: High Availability Design**
- Multi-AZ, load balancing, failover

**Week 4: Cost-Effective Architecture**
- Spot instances, auto-scaling, resource monitoring

---

By the end, you'll be able to architect and deploy systems in the cloud.
