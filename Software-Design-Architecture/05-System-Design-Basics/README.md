# 05 - System Design Basics

Learn the fundamentals of designing scalable, reliable systems. This module covers the building blocks of modern software architecture.

## What You'll Learn

- System design fundamentals
- Scalability and availability
- Database design and tradeoffs
- Caching and CDN basics

## Weekly Breakdown

**Week 1: System Design Fundamentals**
- Load balancers, APIs, databases, caching, rate limiting

**Week 2: Scalability & Availability**
- Vertical vs horizontal scaling
- Availability zones, CAP theorem, eventual consistency

**Week 3: Databases Deep Dive**
- Indexing, partitioning, replication, sharding
- SQL vs NoSQL

**Week 4: Caching & CDN**
- Redis, Memcached, Cloudflare, Akamai

---

By the end, you'll be able to design robust, scalable systems.
