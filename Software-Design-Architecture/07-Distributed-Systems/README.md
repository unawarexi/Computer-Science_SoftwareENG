# 07 - Distributed Systems

Dive into the world of distributed systems. Learn the core concepts, patterns, and challenges of building systems that span multiple machines.

## What You'll Learn

- Distributed computing fundamentals
- Communication patterns (queues, pub/sub, event sourcing)
- CAP theorem and tradeoffs
- Handling failures in distributed systems

## Weekly Breakdown

**Week 1: Core Concepts**
- Distributed computing, time & clocks, consensus

**Week 2: Communication Patterns**
- Message queues (RabbitMQ, Kafka), Pub/Sub, Event sourcing

**Week 3: CAP Theorem in Depth**
- Consistency vs availability
- Design tradeoffs

**Week 4: Failures in Distributed Systems**
- Retry logic, circuit breakers, idempotency

---

By the end, you'll understand how to design and troubleshoot distributed systems.
