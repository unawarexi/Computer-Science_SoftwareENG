# 02 - Software Design Principles

This module dives deep into the essential principles that guide robust software design. You'll learn how to apply these principles to create maintainable, scalable, and flexible systems.

## What You'll Learn

- SOLID principles in depth
- KISS, DRY, YAGNI, Law of Demeter
- How to recognize and avoid anti-patterns
- Refactoring for better design

## Weekly Breakdown

**Week 1: SOLID Principles**
- Deep dive into each SOLID principle
- Real-world examples and exercises

**Week 2: Additional Principles**
- KISS, DRY, YAGNI, Law of Demeter
- When and how to apply each

**Week 3: Anti-Patterns**
- Common design and architecture smells
- How to refactor and avoid them

**Week 4: Practice & Review**
- Refactor old code using learned principles
- Peer review and feedback

---

By the end, you'll be able to design cleaner, more effective software systems.
